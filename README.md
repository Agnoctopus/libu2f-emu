Universal 2nd Factor (U2F) Emulation C Library
==============================================

Libu2f-emu, provide a C Library for the U2F devices emulations.

# API

# Building

Build the binary:
```shell
$ make
```

Test the project:
```shell
$ make check
```

Generate doc:
```shell
$ make doc
```
